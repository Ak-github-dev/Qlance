# Qubic Micro-Freelance Marketplace (Qlance)

A decentralized marketplace on the Qubic blockchain for micro-freelance tasks with:
- Escrow-based payments (no middleman)
- Worker reputation system
- AI-assisted quality verification
- Global accessibility with low fees

## Tech Stack

- **Blockchain:** Qubic (Smart Contracts)
- **Frontend:** React + TypeScript + Tailwind CSS
- **Backend:** Node.js + Express
- **Database:** PostgreSQL (free tier via Render)
- **Hosting:** Vercel (frontend) + Render/Railway (backend)

## Project Structure

Qlance/
├── frontend/ # React dApp UI
├── backend/ # Node.js API server
├── contracts/ # Qubic smart contracts
└── docs/ # Documentation & guides



## Getting Started

### Prerequisites
- Node.js v20+ 
- npm v10+
- Git

### Local Development

Frontend
cd frontend
npm install
npm run dev

Backend (in another terminal)
cd backend
npm install
npm run dev


## Deployment

- Frontend: Vercel (connected to GitHub)
- Backend: Render or Railway (connected to GitHub)
